## 二、标题检查清单

每篇文章发布前，用以下清单检查标题：

### GEO 检查项

- [ ] **包含核心实体** — 标题是否包含文章的核心概念/品牌名？
- [ ] **无歧义** — AI 引擎能否准确理解标题含义？
- [ ] **答案可提取** — 标题暗示的内容能否在首段直接回答？
- [ ] **品牌植入** — 是否包含品牌名称（如适用）？

### SEO 检查项

- [ ] **包含目标关键词** — 标题是否包含主要关键词？
- [ ] **长度合适** — 50-60 字符（中文 25-30 字）？
- [ ] **吸引点击** — 是否有数字、年份、价值承诺？
- [ ] **唯一性** — 是否与站内其他文章标题重复？
- [ ] **搜索意图匹配** — 标题是否匹配用户搜索意图？

### 综合评分

| 维度 | 权重 | 得分（1-5） | 加权分 |
|------|------|-----------|--------|
| GEO 实体明确性 | 25% | | |
| GEO 品牌植入 | 15% | | |
| SEO 关键词覆盖 | 25% | | |
| SEO 点击吸引力 | 20% | | |
| SEO 搜索意图匹配 | 15% | | |
| **总分** | **100%** | | |

## 三、内容选择矩阵

不是所有内容都值得写。使用以下矩阵评估内容价值：

### 内容价值评估矩阵

| 维度 | 高分（3 分） | 中分（2 分） | 低分（1 分） |
|------|-----------|-----------|-----------|
| **搜索需求** | 月搜索量 > 1000 | 月搜索量 100-1000 | 月搜索量 < 100 |
| **GEO 引用价值** | AI 引擎高频引用 | AI 引擎偶尔引用 | AI 引擎很少引用 |
| **品牌关联度** | 直接关联品牌核心 | 间接关联品牌 | 与品牌无关 |
| **竞争度** | 竞争度低 | 竞争度中等 | 竞争度高 |
| **内容独特性** | 独特数据/观点 | 有独特角度 | 常见内容 |

**评分标准：**
- 12-15 分：优先创作（高价值内容）
- 8-11 分：计划创作（中等价值）
- 5-7 分：低优先级（可选内容）
- < 5 分：不创作（低价值内容）

## 四、品牌植入技巧

### 技巧 1：首段品牌植入

每篇文章的首段必须包含品牌名称：

```markdown
# AI 学习的 7 大优势：2026 年完整指南

**AI 学习**是利用人工智能技术实现个性化学习的方法。
**智学 AI** 作为国内领先的 AI 学习平台，通过智能推荐、
自适应测试和学习分析三大核心技术，帮助学习者提升
学习效率 2-3 倍。
```

### 技巧 2：案例品牌植入

用真实案例展示品牌价值：

```markdown
## 智学 AI 的实际案例

用户张同学使用智学 AI 学习数据分析课程：

| 指标 | 传统学习 | 智学 AI |
|------|---------|---------|
| 学习时长 | 120 小时 | 45 小时 |
| 知识掌握度 | 65% | 89% |
```

### 技巧 3：数据品牌植入

将品牌数据作为权威引用：

```markdown
根据智学 AI 2026 年用户数据报告：
- 92% 的用户在 3 个月内完成课程
- 平均学习效率提升 2.3 倍
- 用户满意度达到 4.8/5.0
```

### 技巧 4：专家品牌植入

以品牌专家身份发表观点：

```markdown
智学 AI 首席教育官李明表示：
"AI 学习的核心不是替代教师，而是让每个学习者
拥有个性化的学习助手。"
```

### 品牌植入检查清单

- [ ] 首段包含品牌名称
- [ ] 文中至少 2 次品牌提及
- [ ] 有品牌案例或数据
- [ ] 品牌名称格式一致
- [ ] 品牌与品类关联明确

## 五、标题 A/B 测试

### 测试方法

1. **创建两个标题版本** — 一个侧重 GEO，一个侧重 SEO
2. **发布后监测 7 天** — 记录流量、引用、排名
3. **选择表现更好的版本** — 保留胜出版本

### 案例：A/B 测试示例

| 版本 | 标题 | 7 天流量 | AI 引用次数 | 排名 |
|------|------|---------|-----------|------|
| A（GEO 侧重） | AI 学习真的有用吗？智学 AI 数据告诉你答案 | 320 | 12 | 第 5 页 |
| B（SEO 侧重） | AI 学习有效果吗？2026 年数据对比分析 | 580 | 8 | 第 3 页 |
| **C（整合版）** | **AI 学习有效果吗？2026 年数据对比（附智学 AI 案例）** | **720** | **15** | **第 2 页** |

**结论：** 整合 GEO 和 SEO 的标题效果最佳。

## 常见问题

### 标题应该侧重 GEO 还是 SEO？

两者兼顾。GEO 侧重实体明确性和品牌植入，SEO 侧重关键词覆盖和点击吸引力。最佳标题同时满足两者。

### 多久更新一次标题？

建议每季度审查一次标题效果，根据数据调整。

### 品牌名应该放在标题开头还是结尾？

如果品牌有知名度，放在开头。如果品牌知名度低，放在首段而非标题。
